# Labo DevOps sur Mac M1

Ce projet sert de laboratoire DevOps. Il est conçu pour être exécuté sur un Mac M1 en utilisant une variété d'outils modernes pour la conteneurisation, l'orchestration, le stockage, la surveillance et le déploiement continu.

## Outils Utilisés

*   **Multipass** : Gestion des machines virtuelles (VMs) légères.
*   **K3s** : Distribution Kubernetes légère.
*   **Longhorn** : Solution de stockage distribué pour Kubernetes.
*   **Prometheus** : Système de surveillance et d'alerte.
*   **Grafana** : Plateforme d'analyse et de visualisation.
*   **ArgoCD** : Outil de déploiement continu (GitOps).
